# Huego – Developer Color Palette Tool

## 📌 Overview
Huego is a web-based color palette tool designed to help developers explore, generate, and select color combinations through an intuitive and interactive interface.

The application focuses on clean UI design, responsive interactions, and efficient state management to provide a smooth user experience.

---

## 🚀 Features

- Interactive color palette generation  
- Real-time preview of selected colors  
- Clean and responsive user interface  
- Efficient state handling for smooth updates  

---

## 🛠️ Tech Stack

- TypeScript  
- React 
- CSS for styling and layout  

---
